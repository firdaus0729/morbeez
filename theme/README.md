# Morbeez Shopify Theme (`morbeez-ag`)

Online Store 2.0 theme for agriculture commerce.

## Development

```bash
# From repo root
npm install
npm run build:css
cd theme
shopify theme dev --store YOUR_STORE.myshopify.com
```
